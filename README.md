📖 Overview

GreenWar is an AI-powered sustainability platform designed to help individuals and communities understand, monitor, and reduce their environmental impact. By combining carbon footprint tracking, sustainability analytics, gamified eco-challenges, and AI-driven recommendations, GreenWar empowers users to build eco-friendly habits and contribute to a greener future.

---

🎯 Problem Statement

Climate change and environmental degradation are increasing due to unsustainable daily activities. Most individuals lack awareness of their carbon footprint and do not have accessible tools to track and improve their environmental impact.

GreenWar addresses this challenge by:

- Tracking carbon emissions
- Encouraging sustainable behavior
- Providing AI-powered recommendations
- Promoting community participation
- Gamifying environmental action

📦 Installation

Clone Repository

git clone https://github.com/krishankumawat027-cmd/greenwar.git

Navigate to Project

cd greenwar

Install Dependencies

npm install

Start Development Server

npm run dev

Build Production Version

npm run build

Preview Production Build

npm run preview

---

⚙️ Environment Variables

Create a ".env" file in the root directory.

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
